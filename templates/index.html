How it works: Choose large prime q, primitive root α. Private key x random; public key y=α^x mod q. Encrypt: random k, C1=α^k mod q, C2=M·y^k mod q. Decrypt: s=C1^x mod q, M=C2·s⁻¹ mod q. Sign: S1=α^k mod q, S2=(M-x·S1)·k⁻¹ mod (q-1). Verify: y^S1 · S1^S2 ≡ α^M mod q.

How it works: y²=x³+ax+b over F_p. Non-singular: 4a³+27b²≠0. Point addition via chord-and-tangent. Scalar multiplication via double-and-add. Private key d random; public key Q=d·G. Encryption: each char m → M=m·G, ciphertext (k·G, M+k·Q). Decryption: M=C2−d·C1, recover m from lookup table.
